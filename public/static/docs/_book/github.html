# 分支说明

本项目存在多个分支,每个分支代表的状态是不一样的,根据您的需求下载不同分支

* origin/master:主分支,是稳定版本

* orgin/ver:过度分支,里面是新增功能的测试版,可能存在bug

* origin/dev:开发分支,该分支代码不稳定,是开发中的代码,建议不要拉取

* origin/dev-admin:独立的admin后台分支,主要是后台管理系统
