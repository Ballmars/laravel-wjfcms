# linu权限问题

在安装 Laravel 后，你可能需要配置一些权限。storage和bootstrap/cache目录在你的 web 服务下应该是可写的权限，否则 Laravel 将无法运行。

报错如下:

运行如下命令

sudo chmod -R 777 vendor storage bootstrap/cache
